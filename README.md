# Windows-Cleaner
Utilitário que usa ferramentas do próprio Windows via comandos no terminal para limpeza e otimização do sistema com o objetivo de ser portátil e funcional, sem necessidade de instalação.

Este software portátil executa funções de limpeza e otimização no sistema Windows, incluindo:

- Limpeza de arquivos temporários
- Limpeza de componentes do Windows Update
- Reparo de integridade do sistema
- Otimização de conexão (DNS, IP, cache)
- Otimização da inicialização (desativação de itens desnecessários)

## Como usar

1. Baixe o arquivo ZIP.
2. Extraia tudo para uma pasta.
3. Execute `Utilitário Windows.exe`.
4. Siga o menu exibido no PowerShell.
